1. Чтобы создать БД нужно выполнить команду: rails db:create RAILS_ENV=development.
2. Для создания таблицы + миграции (сразу) нужно выполнить команду: rails g model Question title:string body:text
g - generate, т.е. создать таблицу;
Question - наименование таблицы в БД;
title:string body:text - наименование полей и их типов в БД.
3. Создается файл миграции в db -> <дата>_create_<имя таблицы>.rb, в которой создаются необходимые поля + дополнительные.
4. Создается файл модели.
5. Чтобы применить (совершить) миграцию необходимо ввести команду: rails db:migrate - создается то, что описано в файле миграции.
6. Чтобы откатиться назад необходимо ввести команду: rails db:rollback, доп. параметром (STEP=<количество шагов для отката>) можно указать количество шагов (миграций),
на которые надо откатиться.
7. Чтобы взаимодействовать с таблице исп. модель: app -> models -> questions.rb

Миграция нужна для быстрого развертывания БД на другой машине или сервере (примерно так).
